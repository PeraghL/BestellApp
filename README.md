# BestellApp

## Beschreibung
Die BestellApp ist eine einfache Webanwendung zur Simulation einer Bestellverwaltung.  
Benutzer können Bestellungen eingeben und anzeigen lassen.

Das Projekt dient zur Demonstration von dynamischer Datenverarbeitung im Frontend.

## Funktionen
- Eingabe von Bestelldaten
- Dynamische Anzeige von Bestellungen
- Benutzerinteraktion über Formulare
- Aktualisierung der Oberfläche ohne Seitenreload

## Verwendete Technologien
- HTML5
- CSS3
- JavaScript

## Lernziele
- Verarbeitung von Formulardaten
- Dynamische Erstellung von HTML-Elementen
- Event-Handling
- Strukturierte Programmierung im Frontend

## Projekt starten
Die `index.html` Datei im Browser öffnen.
